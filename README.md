# A pretty blog for hiking storytelling

## Install and develop

```sh
npm install
npm run dev
```

Then head to `http://localhost:3000`. Changes you make to components will be reflected there in realtime.

Deploy:
```
now
```

don't have `now` installed? Find it [here](https://zeit.co/download).
